HTML Basics

What does HTML stand for?

  1. Hyper
  2. Text
  3. Markup
  4. Language

HyperText - text with a bunch of hyper links! Also, HyperMedia (extension of hyper text)

Markup - Content! And how we "markup" content, for example, title tags, div tags, etc.

Language - has its own syntax

Three technologies that drive the Web

  1. HTML - controls the structure
  2. CSS - styles the structure
  3. JS - structure behavior
